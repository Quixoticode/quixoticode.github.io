Was ist Visual Studio Code?

Visual Studio Code (oft als VS Code abgekürzt) ist ein kostenloser, quelloffener und sehr leistungsfähiger Code-Editor, der von Microsoft entwickelt wurde. Er läuft auf Windows, macOS und Linux und hat sich dank seiner Geschwindigkeit, Flexibilität und umfangreichen Feature-Set zu einem der beliebtesten Editoren für Webentwickler und Programmierer weltweit entwickelt.

Kernmerkmale sind:

  • IntelliSense: Intelligente Code-Vervollständigung, Parameter-Infos und mehr.
  • Debugging: Integrierte Debugging-Tools für viele Sprachen, insbesondere JavaScript.
  • Git-Integration: Nahtlose Verwaltung von Git-Repositories direkt im Editor.
  • Erweiterbarkeit: Ein riesiger Marktplatz an Erweiterungen (Extensions), um den Editor an deine Bedürfnisse anzupassen und neue Funktionen hinzuzufügen.
  • Terminal-Integration: Ein eingebautes Terminal, sodass du die Kommandozeile nicht verlassen musst.
  • Hohe Anpassbarkeit: Themes, Tastenkürzel, Einstellungen – fast alles lässt sich konfigurieren.

Installation

Die Installation von VS Code ist unkompliziert:

  1. Besuche die offizielle Download-Seite: VS Code herunterladen
  2. Wähle den passenden Installer für dein Betriebssystem (Windows, Linux .deb/.rpm, macOS).
  3. Führe den Installer aus und folge den Anweisungen. Unter Windows wird empfohlen, die Optionen "Zu PATH hinzufügen" (damit du VS Code aus der Kommandozeile mit code . starten kannst) und "Öffnen mit Code Kontextmenü" zu aktivieren.

Nach der Installation kannst du VS Code starten und mit der Einrichtung beginnen.

Benutzeroberfläche im Überblick

Die Benutzeroberfläche von VS Code ist übersichtlich und in mehrere Hauptbereiche unterteilt:

Stell dir hier ein beschriftetes Screenshot der VS Code Oberfläche vor.

  • Activity Bar (Aktivitätsleiste): Links. Ermöglicht den schnellen Wechsel zwischen Ansichten wie Explorer, Suche, Quellcodeverwaltung (Git), Debugging und Erweiterungen.
  • Side Bar (Seitenleiste): Links neben dem Editor. Zeigt den Inhalt der aktiven Ansicht aus der Activity Bar an (z.B. deine Projektdateien im Explorer).
  • Editor: Der Hauptbereich, in dem du deinen Code schreibst und bearbeitest. Du kannst mehrere Editoren nebeneinander oder untereinander öffnen (Split Editor).
  • Panel: Unten. Kann verschiedene Panels anzeigen, wie das Terminal, die Konsole für Debug-Ausgaben, Probleme (Fehler/Warnungen) und die Ausgabe von Tasks.
  • Status Bar (Statusleiste): Ganz unten. Zeigt Informationen zum aktuellen Projekt, Fehler/Warnungen, Git-Branch, Einrückungseinstellungen, Sprachmodus etc. an.

Grundlegende Bedienung

Dateien und Ordner öffnen

  • Über Datei > Ordner öffnen... (oder Ctrl+K Ctrl+O) öffnest du ein ganzes Projektverzeichnis. Dies ist der empfohlene Weg.
  • Über Datei > Datei öffnen... (Ctrl+O) öffnest du einzelne Dateien.
  • Du kannst Dateien/Ordner auch per Drag & Drop in das VS Code Fenster ziehen.

Neue Dateien/Ordner erstellen

Im Explorer-Panel der Seitenleiste kannst du mit Rechtsklick auf einen Ordner oder einen leeren Bereich "Neue Datei" oder "Neuer Ordner" wählen. Es gibt auch Icons dafür am oberen Rand des Explorer-Panels.

Speichern

Mit Ctrl+S (Windows/Linux) bzw. Cmd+S (macOS) speicherst du die aktive Datei. Mit Ctrl+Shift+S / Cmd+Shift+S speicherst du unter einem neuen Namen.

Command Palette (Befehlspalette)

Eines der mächtigsten Werkzeuge in VS Code! Öffne sie mit Ctrl+Shift+P (Windows/Linux) oder Cmd+Shift+P (macOS).

Hier kannst du nach fast jedem Befehl und jeder Einstellung in VS Code suchen und diese ausführen, ohne dich durch Menüs klicken zu müssen. Beispiele:

  • Preferences: Open User Settings (um Einstellungen zu ändern)
  • Git: Clone (um ein Git-Repository zu klonen)
  • View: Toggle Terminal (um das Terminal ein-/auszublenden)
  • Format Document (um den Code zu formatieren, wenn ein Formatter installiert ist)

Wenn du das > am Anfang der Command Palette entfernst, kannst du direkt nach Dateien in deinem Projekt suchen (entspricht Ctrl+P / Cmd+P).

Wichtige Features für Webentwicklung

IntelliSense & Autovervollständigung

VS Code bietet hervorragende Unterstützung für HTML, CSS, JavaScript und viele andere Sprachen. Während du tippst, erhältst du Vorschläge für Tags, Attribute, CSS-Eigenschaften, JavaScript-Funktionen und -Variablen.

Emmet-Abkürzungen

Emmet ist standardmäßig integriert und beschleunigt das Schreiben von HTML und CSS enorm. Tippe eine Abkürzung und drücke Tab.

HTML Beispiel: ul>li.item$*5{Inhalt $} + Tab wird zu: <ul> <li class="item1">Inhalt 1</li> <li class="item2">Inhalt 2</li> <li class="item3">Inhalt 3</li> <li class="item4">Inhalt 4</li> <li class="item5">Inhalt 5</li> </ul> CSS Beispiel: p10m5 + Tab wird zu: padding: 10px; margin: 5px;

Integriertes Terminal

Öffne es mit Ctrl+` (Backtick, links neben der 1) oder über die Command Palette. Du kannst hier npm-Befehle, Git-Befehle oder andere Kommandozeilen-Tools ausführen, ohne VS Code zu verlassen.

Debugging-Grundlagen (JavaScript)

VS Code hat einen eingebauten Debugger für JavaScript (Node.js und Browser).

  1. Öffne die Debug-Ansicht (Käfer-Icon in der Activity Bar).
  2. Klicke auf "create a launch.json file" und wähle deine Umgebung (z.B. "Web App (Chrome)" oder "Node.js"). Dies erstellt eine Konfigurationsdatei.
  3. Setze Haltepunkte (Breakpoints) in deinem Code, indem du links neben die Zeilennummer klickst.
  4. Starte das Debugging mit F5 oder dem grünen Play-Button in der Debug-Ansicht.
  5. Wenn ein Breakpoint erreicht wird, hält die Ausführung an, und du kannst Variablen inspizieren, Code schrittweise ausführen etc.

Für das Debuggen von Web-Apps im Browser benötigst du oft eine Erweiterung wie "Debugger for Chrome" oder "Debugger for Firefox", oder nutzt die in neueren VS Code Versionen verbesserte Standardkonfiguration.

Quellcodeverwaltung mit Git

VS Code erkennt automatisch Git-Repositories. In der Quellcodeverwaltungs-Ansicht (Verzweigungs-Icon) kannst du:

  • Änderungen anzeigen (Diffs).
  • Änderungen stagen (zum Commit vorbereiten).
  • Commits erstellen und pushen.
  • Branches erstellen und wechseln.
  • Und vieles mehr.

Erweiterungen (Extensions)

Erweiterungen sind das Herzstück der Anpassbarkeit von VS Code. Du kannst sie über die Extensions-Ansicht (Puzzle-Icon in der Activity Bar) durchsuchen und installieren.

Empfehlenswerte Extensions für Webentwicklung:

  • Prettier - Code formatter: Formatiert deinen Code automatisch nach einheitlichen Regeln.
  • Live Server: Startet einen lokalen Entwicklungsserver mit Live-Reload-Funktion für HTML/CSS/JS-Dateien. Sehr nützlich!
  • ESLint: Hilft, Fehler und Stilprobleme in JavaScript-Code zu finden und zu beheben.
  • Path IntelliSense: Autovervollständigung für Dateipfade.
  • Auto Rename Tag: Benennt automatisch das schließende HTML-Tag um, wenn du das öffnende änderst (und umgekehrt).
  • German Language Pack for Visual Studio Code: Für eine deutsche Benutzeroberfläche.
  • Material Icon Theme / vscode-icons: Für schönere Datei- und Ordner-Icons im Explorer.
  • GitLens: Erweitert die Git-Funktionen massiv (Code-Autorenschaft, History etc.).

Installiere nur Erweiterungen, denen du vertraust und die du wirklich benötigst, um die Performance nicht zu beeinträchtigen.

Nützliche Tastenkürzel (Auswahl)

(Windows/Linux-Tasten, für macOS meist Cmd statt Ctrl und Option statt Alt)

  • Command Palette: Ctrl+Shift+P
  • Datei schnell öffnen: Ctrl+P
  • Neue Datei: Ctrl+N
  • Datei speichern: Ctrl+S
  • Alle Dateien speichern: Ctrl+K S
  • Seitenleiste ein-/ausblenden: Ctrl+B
  • Integriertes Terminal ein-/ausblenden: Ctrl+`
  • Zeile(n) nach oben/unten verschieben: Alt+↑ / Alt+↓
  • Zeile(n) duplizieren: Shift+Alt+↑ / Shift+Alt+↓
  • Zeile(n) auskommentieren: Ctrl+/
  • Blockkommentar: Shift+Alt+A
  • Suchen: Ctrl+F
  • Ersetzen: Ctrl+H
  • Multi-Cursor: Alt+Klick (Cursor hinzufügen) oder Ctrl+Alt+↑/↓ (Cursor in Zeile darüber/darunter hinzufügen)
  • Wort markieren und nächste Vorkommen finden: Ctrl+D

Eine vollständige Liste findest du unter Datei > Einstellungen > Tastenkombinationen oder online.

Anpassung

Themes

VS Code kommt mit einigen eingebauten Themes. Weitere Themes (Farbgestaltung für UI und Syntax-Highlighting) kannst du über die Extensions-Ansicht installieren.

Wechsle Themes über Datei > Einstellungen > Farbdesign (oder Ctrl+K Ctrl+T).

Einstellungen (Settings)

Fast jeder Aspekt von VS Code lässt sich anpassen. Öffne die Einstellungen:

  • Über die UI: Datei > Einstellungen > Einstellungen (oder Ctrl+,)
  • Als JSON: Öffne die Command Palette (Ctrl+Shift+P) und suche nach Preferences: Open User Settings (JSON).

Beispiele für nützliche Einstellungen:

{ "editor.fontSize": 14, "editor.tabSize": 2, "editor.wordWrap": "on", // Zeilenumbruch aktivieren "files.autoSave": "afterDelay", // Automatisches Speichern "files.autoSaveDelay": 1000, // Nach 1 Sekunde Inaktivität "editor.formatOnSave": true, // Automatisch formatieren beim Speichern (Formatter muss konfiguriert sein) "workbench.iconTheme": "vscode-icons", // Falls "vscode-icons" installiert ist "[javascript]": { // Sprachspezifische Einstellungen "editor.defaultFormatter": "esbenp.prettier-vscode" } }
